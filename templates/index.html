Text Processing Demonstrator


This simple text processing demonstrator takes input from two sources: either a web page or a PDF file.

If a web page URL is submitted, it is scraped on demand using Playwright to extract the text.

If a PDF file is submitted then PDF Plumber is used to extract the text.

Once submitted, the text is processed using KeyBERT, which processes it to establish the most representative keywords, shown on the following page.
